# GCE Features: Custom Machine Types

When predefined VM options are **not appropriate** for our workload, we are able to create a machine type customized to our needs (a `Custom Machine Type`).

With selecting Custom type, we can adjust **number of vCPUs**, **memory** and **GPUs**.

- We can choose between `E2`, `N2`, or `N1` machine types
- Custom machines support variety of Operating Systems: `CentOS`, `CoreOS`, `Debian`, `Red Hat`, `Ubuntu`, `Windows` etc.
- We are billed per vCPUs, memory provisioned to each instance
  - **Example**: Hourly price $0.033174/vCPU + $0.004446/GB
