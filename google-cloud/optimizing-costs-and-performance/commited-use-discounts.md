# Commited Use discounts in GCP

Using commited use discounts is recommended for workloads with **predictable resource needs**.

They are applicable for instances created by **Google Kubernetes Engine** and **Compute Engine**

> [!NOTE]
> **Requirement**
>
> - **Register/Commit** for VM for 1 year or 3 years usage to obtain commited use discount
> - Up to **70% discount** based on machine type and GPUs
> - Discount of using commited use discounts `is higher` than discount by sustained use discounts

> [!WARNING]
> **RESTRICTIONS**
>
> - Does not apply to VMs created by **App Engine flexible** and **Dataflow**
